# Simple Push Command

Copy and paste this command, replacing `YOUR_TOKEN_HERE` with your actual GitHub token:

```powershell
git remote set-url creator-talent-hub https://YOUR_TOKEN_HERE@github.com/scabadoobop/creator-talent-hub.git && git push -u creator-talent-hub main
```

**Example** (don't use this exact token, use yours):
```powershell
git remote set-url creator-talent-hub https://ghp_abc123xyz@github.com/scabadoobop/creator-talent-hub.git && git push -u creator-talent-hub main
```

After running this, your code will be pushed to GitHub!
